# A-simple-chess-interface-that-I-will-try-to-improve
This was one of my first bigger applications, it is a little old and not perfectly optimized, but I will improve on it in the future. 
# How to use
Use the same code as in the designer file to set up the board. Everything else works as normal chess, but you can also set up your own board with the following syntax by entering it in the text file. First line is for white and second one is for black. P1222324252627282 N2171 B3161 R1181 Q41 K51 . The order does not matter. You can even decide to have several kings if you want, it could be really fun. Just like in real chess, if you are in check, no one is going to tell you and there will be no indication, this is left like that on purpose. If you don't see the check, you lose the game.
